Projeto feito no Intelij IDEA.
Jdk 10.0

Abrir projeto no Intelij IDEA;
Clicar em File -> Project Structure;
Selecionar a opção Artifacts e clicar no símbolo + verde -> "JAR" -> From module with dependencies;
Selecionar a Main Class como Planarity -> OK;
Na Menubar do IDEA selecionar Build -> Build artifacts -> PlanarityPuzzle.jar -> Build;
O ".jar" estará na pasta PlanarityPuzzle\out\artifacts\PlanarityPuzzle_jar.
